Prompt Editor Help ({{ template_safe_prompt_id }})

This page allows you to edit the content, metadata, and view dependencies of a prompt.

Keyboard Shortcuts
ShortcutAction
Alt + SSave prompt content and metadata
Alt + CCopy prompt content with inclusions resolved to clipboard
Alt + Shift + CCopy rendered prompt content to clipboard
Alt + TToggle between Raw and Expanded content views
Alt + ROpen Search/Replace dialog (in Raw Content view)
Alt + BGo back to the prompt list

Alt + H or click Show this help dialog
Functionality
Main Editor Area:
  • Raw Content Tab: Edit the direct content of the prompt. Use [[prompt_name]] to include other prompts (case-sensitive).
  • Expanded Content Tab: View the prompt with all inclusions fully expanded (read-only). This is how the prompt will be seen by models/tools that consume it.
  • The editor auto-saves content and metadata changes shortly after you stop typing. The "Save" button provides immediate manual saving.
Sidebar:
  • Metadata Card:
    • Type Badge: Automatically updates to 'Standard' or 'Composite' based on whether [[...]] inclusions are present in the raw content.
    • Last Updated: Timestamp of the last save.
    • Directory: The directory this prompt belongs to.
    • Description: A brief, editable description for the prompt.
    • Tags: Comma-separated tags for categorizing and searching.
  • Dependencies Card: Lists all other prompts directly included by this prompt (i.e., prompts referenced as [[dependency_name]]). Click to navigate to the dependency.
  • Referenced By Card: Lists all other prompts that include this prompt. Click to navigate to the referencing prompt.
Header Buttons:
  • Back to List: Navigates back to the "Manage Prompts" page.
  • Copy: Copies the prompt content with all inclusions resolved but preserves the original markdown format.
  • Copy Rendered: Copies the prompt content as rendered text (Alt+Shift+C).
  • Toggle View: Switches between the "Raw Content" and "Expanded Content" tabs.
  • Search/Replace: Opens a search and replace dialog for the "Raw Content" editor.
  • Rename: Allows you to change the ID (name) of the prompt. This will also update references in other prompts if possible.
  • Save: Manually saves any pending changes to content or metadata. The button provides feedback during the save process.
  • Delete: Deletes the prompt (with confirmation).
